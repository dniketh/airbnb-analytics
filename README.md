A dbt Project for Modelling and Transforming Airbnb Data from May 2020 to April 2021 to generate insights 

### Use dbt Cloud and Run the following Commands
- dbt build - To build bronze, silver, snapshots and gold layer after loading the data via Airflow GCP Composer
- dbt test  - To Test Schema files to ensure the tables follow the conditions in place for transforming the data
